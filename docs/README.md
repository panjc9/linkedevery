## 文档命名规范

1. **中文文档**：使用中文名称，如 `TRL-IRL深度研究报告.md`
2. **英文文档**：使用英文名称，如 `README.md`
3. **版本控制**：在文件名中体现版本，如 `方案v2.docx`
4. **日期标记**：重要文档可添加日期，如 `2026-06-16-报告.md`
